Basic I/O

cin & cout

cout is used for output. It uses the insertion opertaor <<

        
          cout << "Hello World";  // prints string literal "Hello World"
          cout << 123;      // prints integer constant 123
          cout << x;        // prints the value of the variable x

These two ouputs has different results

        
            cout << "hello";    // prints the string literal "hello"
            cout << hello;      // prints the value of the variable hello

Chaining

Use multiple << to do chaining

        
          cout << "Hello" << " my name is " << "Roberto.";

You can mix literals and variables in CHAINING

        
          string name = "Roberto";  
          cout << "Hello" << " my name is " << name;      //prints "Hello my name is Roberto"

cout does not add a new line automatically. Add a new line by chaining << endl or adding \n within the string. 

        
        cout << "This is first line" << endl;     // using endl
        cout << "This is second line\n";        // using newline character
        cout << 123 << '\n';              // chaining with newline character
        cout << "One line . \n Another Line";     // using newline within string

cin

cin is used to extract data from an input and store that data into a variable. It uses the extraction operator >>

        
          int age;
          cin >> age;

cin uses the datatype of the variable after >>. In the above example, cin is expecting an int. If the input datatype is different, cin will fail.

cin can also be chained in one extraction of more than one data in a single statement

        
          cin >> a >> b;

          //also the same as 
          cin >> a;
          cin >> b;

cin and strings

cin can be used to get strings of characters, but only gets those before a space.

        
        string name;  //Roberto Rodriguez
        cin >> name;  
        cout << name; //Roberto

getline()

this function gets the entire line from input.

        
          //format
          getline(cin, variable);
        
      
        
          string name;
          getline(cin, name);

          //invalid. getline is for strings only
          float grades;
          getline(cin, grades);

More on getline()

Notice how the variable name is used multiple times in getline? The second call will just replace whatever is the existing value of name

        
          string name;
          cout << "What is your name : " << endl;
          getline(cin, name);

          cout << "What is your nickname : " << endl;
          getline(cin, name);

stringstream() parses a value from a string depending on the datatype of the variable it is storing values to.
#include < sstream > must be added at the top 

        
          string myString;
          float price;

          cout << "Enter price";    //user enters 50
          getline(cin, myString);   //50 is stored in mystring, myString = "50";
          stringstream(myString) >> price;

When user enters 50, 50 will then be stored in the variable myString as a string. stringstream will then parse myString and will examine if there is a valid value as float. That value will then be stored to the variable price as a normal float.

Switch Case

Syntax

       
       switch(expression)
        {
          case constant1:
               group-of-statements-1;
               break;
          case constant2:
               group-of-statements-2;
               break;
          case constant3:
               group-of-statements-3;
               break;
          default:
              default-group-of-statements
        }

switch evaluates the expression first then checks if it is equivalent to a certain constant.

If the expression is equivalent to constant1, it will run the statements inside it until it finds break

If there are no equivalent constant, it will run statements in default

Example

In the code below, case 2 is evaluated because x = 2

            
            int x = 2;

            switch(x)
            {
              case 1:
                cout << "x is 1" << endl;
                break;
              case 2:
                cout << "x is 2" << endl;
                break;
              case 3: 
                cout << "x is 3" << endl;
                break;
              default:
                cout << "x is unknown" << endl;
            }

            // output "x is 2"

case label expressions must be unique values

            
            switch(x)
            {
              case 4:
                statement;
                break;
              case 4:         //illegal because it is not unique
                statement;
                break;
              default:
            }

break

break statement tells the compiler that we are done with a particular switch, and execution continues with the statement after the end of the switch block

      
        int x = 1;
        switch(x)
        {
          case 1:
            printf("This is case 1\n");   // no break statement

          case 2:
            printf("This is case 2\n");
            break;
          default:
            printf("This is the default\n");

        }

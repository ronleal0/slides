Return Value

Functions you write can return a single value to their caller as well. We do this by changing the return type of the function in the function’s declaration. A return type of void means the function does not return a value. A return type of int means the function returns an integer value to the caller. 

        
        // int means the function returns an integer to the caller
        int Return4(){
          return 4;
        }

        // void means the function does not return anything
        void ReturnNothing(){
          cout << "Dont return anything" << endl;
        }

constant parameters are essentially values that dont actually change within the scope of the function

function definition vs function declaration

function definition is the complete creation of function prototype. It includes the header and the body of the function. function declaration is just a normal invoking of the function which includes return type and the identifier (name of the function)

      
       // function declaration
       int add();
  
       int main(){
          add(1,1);

          return 0;
        }

        // function definition
        int add(int num1, num2){
          return num1 + num2;
        }

Dedication of Duty

As much as possible, a function should only do a singular task and no more. 

      
        // function to get average of 2 values
        float average(float value1, float value2)
        {
          float sum,ave;

          sum = value1 + value2;
          ave = sum/2;
          return ave;
        }

... code above can still be broken down

      
        float getSum(float value1, float value2)
        {
            return value1 + value2;
        }

        float average(float value1, float value2)
        {
          return getSum(value1, value2)/2;
        }

Sometimes you need to setup a default parameter in case the argument is not provided

      
        // function declaration
        void print_Given_Number(float number = 5);

        int main(){

          print_Given_Number(3); // will print 3
          print_Given_Number(2); // will print 2
          print_Given_Number(1); // will print 1
          print_Given_Number(); // will print  5 because the default value is 5

          return 0;
        }

        // function definition
        void print_Given_Number(float number){
          cout << number << endl;
        }

If a parameter has a default value, the next parameters should also have a default value

      
        int add(int a = 5, int b = 2);    // will work
        int add(int a, int b = 5);         // will work
        int add(int a = 5, int b);         // will not work

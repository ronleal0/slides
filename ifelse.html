Selection Structure

if else

if else condition

Basic syntax

             
             if(condition)
                statement

In the example below, if number is not equal to 100, the statement is ignored

             
             if( number == 100)
                cout << "number is 100";

if you want to include more than 1 statement, use the scope operators {} to form a block/scope

       
       if( number == 100){
          cout << "number is 100";        // first statement
          cout << "This is another statement" ;   // another statement
        }

indentation and line breaks has no effect so long as statements are properly terminated, you can write like this:

       
       if(number == 100){cout << "number is 100"; cout << "This is another statement";}

use else for alternative statements

       
       if(condition)
          statement
       else
          alternative statement

example

       
      if( number == 100 ){
        printf("it is 100");
      }else{
        printf("it is not 100");
      }

Several conditions can also be tested using else if 

      
      if( age >= 7)                         // first condition
      { 
        cout << "A kid" << endl;
      }else if((age >= 8) || (age <= 17))   // another condition
      {
        cout << "A teen" << endl;
      }else if((age >=18 ) || (age <= 30))  // another condition
      {
        cout << "Young Adult" << endl;
      }else
      {
        cout << "Adult" << endl;
      }

Nested condtions

You can also nest condtions inside another conditions

             
             if(gender == "boy"){

              //check if old or young
              if(age < 18){
                cout << "You are a young boy";
              }else{
                cout << "You are an old boy";
              }
            }else{

              //check if old or young
              if(age < 18){
                cout << "You are a young girl";
              }else{
                cout << "You are an old girl";
              }
            }

Ternary Operation

Ternary operation is only applicable for simple condition. 1 statement per block

       
       if (score < 75){
        printf("You failed");
       }else{
        printf("You passed"); 
       }

The above statement can also be written like this (in ternary operation)

       
        (score < 75 ) ? printf("You failed") : printf("You passed");

        //since it is an operation, you can store result to a variable
        string result;
        result = (score < 75 ) ? printf("You failed") : printf("You passed");
